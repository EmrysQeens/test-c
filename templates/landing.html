The definition of coding is the process of creating instructions for computers using programming languages. Computer code is used to program the websites, apps, and other technologies we interact with every day.

There are lots of languages which are graphical, text based, blocks and a lots more.

  • Scratch : This is a graphical programming language designed by MIT which uses blocks, sprites, stages for its programming. The graphical aspects makes it fun and childrens easily grap the basic idea of programming even though it has limited usage.
  • C : This is a text based old programming language used for digital electronics, creating of operating systems such as windows, linux, Database systems, Graphics packages, Word processors, Spread sheets, Compilers and Assemblers, Network drivers, Interpreters.
  • Python : This is a text based language used for Artificial Intelligence, Web applications, Finance, Data analysis and lots more.
  • CSS (Cascading StyleSheet) : This is a text based langugae used for colors, styling html elemnts used on the web. It defines the attributes of elements in a web page.
  • Javascript : This is a text based language used in the browser for making web pages user interactive. Odds are you have played games on browsers such as chess, 2024, Tic Tac Toe, e.t.c are all made interactive with this language
  • HTML (Hypertext Markup Language) : This is an XNL (tag based language) used for layout of the web pages displayed in browsers.

The world is rapidly changing and one of the main cause is rapid changes in science.

Science is the study of the world and how it is rapidly changing around us. Scientist observes, describe, experiment. there ae different branches of science, which are classified based on what they study such as geology, biology, physics, chemistry.

  • Biology : This is the study of life and also of living organisms. The word is coined from two greek words "bios", 'logia", where bios means life and logia means study of. Experts in this field are called Biologist.
  • Physics : This is a branch is science that studies matter, motion, energy, forces, e.t.c and how they interact with one another. Experts in this field are called Physicist. Some famous physicist in history are Albert Einsten, Issac Newton ..........
  • Chemistry : This is a branch of science that studies properties of matter. Experts in this field are called Chemists. Chemistry is all around us and it is in the drugs doctors prescribe, electronics like Tv's, Mobile phones ...., Also on the farm as farmer grow crops (Fertilizer) and a lot more.
